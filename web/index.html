Vous devez spécifier le nom de l'invocateur		 /rank?region=NA1&summonerName=test ICI

Liste des paramètres possibles

Paramètres Obligatoire Description Valeur disponible Exemple
region OUI Spécifier le serveur ou vous jouez NA1, EUW1 /rank?region=NA1
summonerName OUI Le nom de l'invocateur /rank?region=NA1&summonerName=Bedy90
winRate NON Afficher le % victoires 0 ou 1 /rank?region=NA1&summonerName=Bedy90?winRate=0
lp NON Afficher ou non les LP 0 ou 1 /rank?region=NA1&summonerName=Bedy90?lp=0
series NON Caractères de remplacer pour les Wins/Losses/Not Play /rank?region=NA1&summonerName=Bedy90?series=XYZ
queueType NON Définir la queue pour laquel on désire obtenir le rang solo5, tft /rank?region=NA1&summonerName=Bedy90?queueType=tft
